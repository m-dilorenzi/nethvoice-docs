Questa sezione consente l'accesso al pannello di amministrazione FreePBX sottostante e ad altre opzioni di configurazione avanzate.

:::warning Prerequisiti
Assicurati di avere familiarità con la [configurazione di base di NethVoice](../configuration/wizard.md) prima di procedere con le impostazioni avanzate. Le configurazioni non corrette in questa sezione potrebbero influire sulla stabilità del sistema.
:::

:::info Supporto
Per assistenza con configurazioni avanzate, consulta la [documentazione ufficiale di FreePBX](https://wiki.freepbx.org/) o contatta il supporto NethVoice.
:::

### Quando utilizzare l'interfaccia avanzata

Accedi all'interfaccia Avanzata quando hai bisogno di:

- Configurare impostazioni FreePBX avanzate non disponibili nella procedura guidata principale di NethVoice
- Gestire configurazioni a livello di sistema
- Accedere a strumenti di reporting e monitoraggio
- Eseguire manutenzione e risoluzione dei problemi del sistema
- Configurare interni personalizzati e applicazioni

### Best Practice

- Fare sempre un backup del sistema prima di apportare modifiche alla configurazione avanzata
- Documentare tutte le configurazioni personalizzate per riferimenti futuri
- Testare le modifiche di configurazione in un ambiente di sviluppo se possibile
- Consultare la documentazione di FreePBX per attività di configurazione complesse

## Impostazioni finali

La procedura guidata è quasi completa. Ora puoi configurare alcuni parametri finali prima di accedere all'interfaccia NethVoice CTI:

- **Imposta la lingua del sistema**: il valore predefinito è l'inglese, ma puoi cambiarla con la tua lingua preferita.
- **Cambia password**: per motivi di sicurezza, è consigliabile cambiare la password di amministrazione predefinita.
